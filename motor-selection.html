ME541: Underwater Automated Leveling Table for Ultrasonic NDE

Introduction

This project focuses on selecting linear actuators for an underwater automated leveling table for ultrasonic nondestructive evaluation (NDE). This project was to provide some background research for the senior design where the goal is to automate the alignment of sample surfaces for ultrasonic transducers with high precision (≤15 µm) while operating underwater. The system requires motorized, adjustable legs and feedback control for leveling.

Decision Situation

The core decision is selecting the best design alternative for the leveling mechanism. The system must operate effectively underwater, provide precise adjustments, and stay within a $5,000 budget. Design alternatives include different actuators (linear, stepper motors, pneumatic systems) and materials (corrosion-resistant alloys, waterproof coatings).

Framing the Decision

The decision is framed around four fundamental objectives:

  • Maximize Precision: Achieve alignment accuracy of ≤15 µm.
  • Maximize Durability: Ensure reliable operation in underwater environments.
  • Minimize Cost: Stay within the $5,000 budget.
  • Maximize Ease of Use: Simplify system operation for researchers.

Attributes and Alternatives

Key attributes include:

  • Motor Speed (RPM)
  • Response Time (ms)
  • Corrosion Resistance (Rating)
  • Price ($)
Three alternatives were evaluated:
  • M-228.10S Linear Actuator: Compact, precise, and durable.
  • LAS10 Linear Servo Actuator: High speed and accuracy.
  • XLA-5 Series Linear Actuator: Exceptional speed and precision.

Single-Attribute Utility Functions

Utility functions were developed for each attribute:

  • Response Velocity: Exponential utility function.
  • Response Accuracy: Logarithmic utility function.
  • Corrosion Resistance: Piecewise utility function.
  • Price: Third-order polynomial utility function.

Multi-Attribute Utility Function

A multi-attribute utility function was developed using scaling constants: 

u(V,A,C,P) = (4.5e-7)u1(V) + (0.74685)u2(A) + (0.1671)u3(C) + (0.0853)u4(P)

Expected Utility Evaluation

The expected utility for each alternative was calculated:

  • M-228.10S: 0.92718
  • LAS10: 0.73756
  • XLA-5 Series: 0.87918
The M-228.10S was identified as the most preferred alternative due to its high expected utility.

Discussion and Learning

This project highlighted the importance of balancing technical performance and cost while considering uncertainty in input data. The structured decision-making approach provided a clear framework for evaluating alternatives in complex systems. This approach can be applied to future projects involving multiple conflicting attributes.
