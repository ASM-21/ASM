ME 375: Skitter Robot - Autonomous Line and Hand Following

Introduction

The goal of this project is to program the Skitter Robot to perform two autonomous tasks:

  1. Follow a line around a 320-inch track.
  2. Maintain a nine-inch distance from an operator's hand while following it in a straight line.
The robot utilizes subsystems such as encoders, an ultrasonic sensor, a line-following sensor, two motors, a gearbox, and an Arduino for control.

Subsystem Characterization

Each subsystem was tested to understand its performance:

  • Encoders: Measured distance traveled by relating wheel rotations to encoder counts.
  • Ultrasonic Sensor: Tested for distance measurement accuracy, with a nonlinear relationship between sensor values and distance.
  • Line-Following Sensors: Binary sensors detect the line position, with a combined output for precise tracking.
  • Motors: Friction compensation and system identification were performed to improve control.

Controller Design

A PI (Proportional-Integral) controller was designed for closed-loop feedback control. The transfer function for the motors was derived, and a controller was calculated to achieve a settling time of 0.3 seconds and 10% overshoot.

The final controller is: 

C(s) = 1.544 + 73.39/s

State Machines

Two state machines were developed:

  • Line-Following State Machine: Adjusts the robot's motion to follow the line, with states for forward motion, sharp turns, and reversing if the line is lost.
  • Hand-Following State Machine: Uses the ultrasonic sensor to maintain a nine-inch distance from the operator's hand.

Potential Problems

Challenges included:

  • Accurate system identification for the plant function.
  • Variability in sensor performance due to environmental factors.
  • Ensuring smooth transitions between states in the state machines.
