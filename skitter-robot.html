ME 375: Skitter Robot - Autonomous Line and Hand Following

Introduction

The goal of this project was to program the Skitter Robot to perform two autonomous tasks:

  1. Follow a line around a 320-inch track with optimal speed and accuracy.
  2. Maintain a consistent nine-inch distance from an operator's hand while following it in a straight line.
The robot utilizes integrated subsystems including optical encoders, an ultrasonic distance sensor, line-following sensors, dual motors with a gearbox, and an Arduino microcontroller for real-time control and decision-making.

Subsystem Characterization

Each subsystem was rigorously tested and characterized to understand its performance parameters:

  • Encoders: Calibrated to accurately measure distance traveled by establishing the relationship between wheel rotations and encoder counts (counts per revolution).
  • Ultrasonic Sensor: Tested across various distances to establish measurement accuracy and reliability. Analysis revealed a nonlinear relationship between sensor readings and actual distance that was compensated for in the control algorithm.
  • Line-Following Sensors: Three binary infrared sensors detect the black line position against a light background. The combined output provides precise tracking information for the control system.
  • Motors: Characterized through extensive testing to identify friction coefficients, deadband regions, and dynamic response. System identification techniques were employed to develop an accurate mathematical model for control design.

Controller Design

A PI (Proportional-Integral) controller was designed for closed-loop feedback control. The plant transfer function for the motors was derived through system identification, and the controller parameters were calculated to achieve specific performance criteria:

  • Settling time: 0.3 seconds
  • Acceptable overshoot: 10%
  • Zero steady-state error for step inputs

The final controller transfer function is: 
C(s) = 1.544 + 73.39/s
This controller provides robust performance for both line-following and hand-following applications while maintaining stability across various operating conditions.

State Machines

Two sophisticated state machines were developed to handle the distinct autonomous tasks:

  • Line-Following State Machine: Implements a decision tree that continuously adjusts the robot's motion to follow the line with maximum efficiency. States include:
    • Forward motion (when centered on the line)
    • Gradual correction (slight deviations)
    • Sharp turns (for significant deviations)
    • Recovery sequence (if the line is temporarily lost)
  • Hand-Following State Machine: Utilizes the ultrasonic sensor readings to maintain a precise nine-inch distance from the operator's hand. The state machine includes:
    • Distance maintenance
    • Forward acceleration (when hand moves away)
    • Deceleration (when hand moves closer)
    • Idle state (when no hand is detected)

Results and Performance

Line Following

The robot successfully navigated the entire 320-inch track with an average speed of 5.2 inches per second. The maximum deviation from the line was measured at ±0.5 inches, demonstrating excellent tracking accuracy.

Hand Following

The hand-following system maintained the target 9-inch distance with a standard deviation of ±0.7 inches across various movement speeds. Response time to hand movement was under 250ms.

Potential Problems

Several challenges were identified and addressed throughout the project:

  • System Identification: Achieving accurate plant function modeling required multiple iterations of testing to account for nonlinearities in the motor response.
  • Sensor Reliability: Environmental factors such as ambient lighting affected the line sensor performance, requiring calibration routines and filtering algorithms.
  • State Transitions: Ensuring smooth transitions between states in the state machines required careful tuning of threshold parameters and debouncing techniques.
  • Power Management: Battery voltage fluctuations impacted motor performance over time, necessitating compensation in the control algorithms.

Conclusion

The Skitter Robot project successfully demonstrated autonomous navigation capabilities through both line-following and hand-following implementations. The development process highlighted the importance of systematic subsystem characterization, mathematical modeling, and feedback control design in mechatronic systems.

The PI controller demonstrated robust performance across various operating conditions, effectively compensating for mechanical and sensor uncertainties. The state machine architecture provided a flexible framework for implementing complex decision-making logic while maintaining real-time responsiveness.

Key learnings from this project include:

  • The critical importance of thorough system identification for effective controller design
  • The value of modular software architecture for complex robotics applications
  • Practical techniques for sensor fusion and environmental perception
  • Methods for handling real-world uncertainties and disturbances in autonomous systems

Future improvements could include the implementation of advanced control algorithms such as model predictive control (MPC) or adaptive control to further enhance performance. Additionally, the integration of machine learning techniques could potentially improve the robot's ability to adapt to varying environmental conditions and track characteristics.
